Design Principles

One Accent Color
Cyan (#00d4ff) is the only brand color. Everything else is grayscale or semantic (green=success, red=danger). No competing colors.
Layered Depth
4 background shades create depth without borders. Base → Sidebar → Surface → Elevated. Subtle, not heavy.
Terminal First
Chrome fades away. The sidebar, header, and tabs are subtle — the terminal content gets all the visual weight.
Consistent Borders
All borders use #243447. One color for all dividers, separators, and outlines. No visual noise.
